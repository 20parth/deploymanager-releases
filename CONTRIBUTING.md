# Contributing to DeployManager

Thank you for your interest in contributing!

## Ways to contribute

- **Bug reports** — Open an [issue](https://github.com/20parth/deploymanager-releases/issues) with steps to reproduce
- **Feature requests** — Open an issue describing what you'd like and why
- **Screenshots / GIFs** — Share demos in issues or discussions
- **Documentation** — Fixes and improvements to README or guides

## Reporting a bug

Please include:
1. OS and version (e.g. Ubuntu 22.04)
2. Install method (Docker or bare metal)
3. Steps to reproduce
4. Expected vs actual behaviour
5. Relevant logs (`pm2 logs deploymanager` or `docker compose logs backend`)

## Security vulnerabilities

Do **not** open a public issue for security bugs. Instead open a **private security advisory** on GitHub.
