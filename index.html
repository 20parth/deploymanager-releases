Features

Everything you need to ship faster

Built for freelancers and small teams who want CI/CD without the SaaS pricing.

🔄
Git-triggered deploys
Push to any branch and DeployManager runs your deploy script automatically via GitHub or GitLab webhooks.
📊
Real-time log streaming
Every stdout/stderr line streams live to your browser via Socket.io. Watch builds as they happen.
🏗️
7 framework templates
React/Vite, Vue/Vite, Node.js, PHP, Python, Static, and fully custom bash scripts.
🔐
Secure webhooks
HMAC-SHA256 signature verification on every incoming request. No unauthenticated triggers.
👥
Role-based access
ADMIN, DEVELOPER, and VIEWER roles. Control who can trigger deploys and who can only watch.
🌿
Multi-project & environments
Unlimited projects. Separate staging, production, and preview environments per project.
🐳
Docker or bare metal
Run in containers from Docker Hub or install directly on the host with Node + PM2.
📝
Script editor
CodeMirror 6 bash editor with syntax highlighting and autocomplete snippets built in.
🏠
Works with any hosting panel
Compatible with cPanel, CloudPanel, Plesk, HestiaCP, CyberPanel, and plain VPS setups.

Install

Up and running in under 5 minutes

One command installs everything — PostgreSQL, Nginx, SSL, and the app.

# Downloads and runs the interactive installer
curl -fsSL https://20parth.github.io/deploymanager-releases/install.sh | sudo bash

# The installer asks:
#   Domain or IP  →  deploy.example.com
#   Admin email + password
#   Install method  →  1) Bare metal  2) Docker
#   Using a hosting panel?  →  y/n
#   Enable HTTPS?  →  y/n
# Pull and run from Docker Hub
docker pull parthrb/deploymanager-api:latest
docker pull parthrb/deploymanager-web:latest

# Or use the installer and choose option 2
curl -fsSL https://20parth.github.io/deploymanager-releases/install.sh | sudo bash
curl -fsSL https://20parth.github.io/deploymanager-releases/docker-compose.yml -o docker-compose.yml

# Create backend/.env with your settings
mkdir backend
cat > backend/.env <<EOF
DATABASE_URL="postgresql://deploymanager:secret@postgres:5432/deploymanager"
JWT_SECRET="$(openssl rand -hex 64)"
JWT_REFRESH_SECRET="$(openssl rand -hex 64)"
PORT=5000
NODE_ENV=production
CORS_ORIGIN="https://your-domain.com"
WEBHOOK_BASE_URL="https://your-domain.com"
EOF

docker compose up -d
